# Telegram Stamp Bot

Simple Telegram bot for collecting stamps from station QR codes.

## Install

1. Install Python 3.
2. Install the Telegram bot library: (better option: alternatively run pip install -r requirements.txt)

```bash
pip install python-telegram-bot
```
## Run

```bash
python bot.py
```

To stop the bot, press `Ctrl + C`.

## Test

In Telegram, send:

```text
/start
```

To test a station stamp:

```text
/start station1
```
